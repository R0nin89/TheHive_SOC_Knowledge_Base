# Glosario SOC

| Término | Significado |
|---|---|
| IOC | Indicador de compromiso: IP, dominio, URL, hash, email, usuario o host |
| TTP | Táctica, técnica y procedimiento usado por un atacante |
| SIEM | Plataforma de correlación y análisis de eventos |
| EDR | Telemetría y respuesta en endpoint |
| NDR | Telemetría y detección en red |
| IDS | Sistema de detección de intrusiones |
| Falso positivo | Alerta legítima o no maliciosa |
| Verdadero positivo | Actividad maliciosa o no autorizada confirmada |
| Toma de cuenta | Uso no autorizado de credenciales válidas |
| Password spraying | Pocas contraseñas probadas contra muchos usuarios |
| Brute force | Muchos intentos contra una o varias cuentas |
| LOLBin | Binario legítimo usado con fines maliciosos |
| C2 | Canal de mando y control |
| Beaconing | Comunicación periódica hacia infraestructura externa |
| Exfiltración | Salida no autorizada de datos |
| Persistencia | Mecanismo para mantener acceso |
| Evasión defensiva | Acción para ocultar actividad o desactivar controles |
